# GraphMHC
Neoantigen prediction model applying the graph neural network to molecular structure from amino acid sequences of MHC proteins and peptides

## Requirements
* python=3.9
* pandas=1.4.4
* numpy=1.23.2
* rdkit=2022.03.2
* torch=1.11.0+cu113
* torch_geometric=2.1.0
* sklearn=1.1.2

## Running
```
python graphmhc.py
```
